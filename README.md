# Real-time-Analytics

A scalable distributed system that ingests high-volume user events,
processes them in real time, and exposes analytics through APIs and dashboards.

## Key Features

- Event streaming architecture
- Real-time analytics
- Distributed processing
- High throughput ingestion
- Observability and monitoring

## Tech Stack

Kafka
Redis
ClickHouse
Docker
Python/Node.js
Grafana
Prometheus
